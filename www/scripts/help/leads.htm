Form Control Buttons

Browse

Use this button to find the lead file using your local filesystem. When selected, the file will be displayed to the left of the button.

Upload

Instructs the browser to upload the displayed file from the workstation to the dialer. See also, "Check Nodial List" and "Enable Mobile" below.

Check Nodial List

When checked, filters the uploaded file, at upload time, against the nodial lists present in the Customer's "Nodial" section.

Enable Mobile

When checked, allows mobile/Cell phone number to be dialed by the system.

Download

Used to download the selected file to the local workstation.

Delete

Removed the selected file from the Dial file list and removes it from the system.

Order New Leads

Opens a form to order additional leads for the project.
