Voice


Displays a list of the current Voice files used by the project.

Voice files are added to a project through uploading, or a "Copy to Project" process. Voice files can be recorded using the dial-in recording features. Recorded messages are digitally converted and then stored on the Voice server in the Customer Files section.

In this form, the Customer files are viewed on the right side of the form. Files currently assigned to the project are viewed on the left. Voice prompt files are copied from the Customer files to the project by first highlighting the voice file on the right, and then clicking on the "Copy to Project" button.

Files can also be copied from the project to the Customers folder on the Dialin Voice Server using the "Copy to Customer".

Form Control Buttons

Copy to project

Copies the selected voice prompt file from the Customer files to the Project voice prompts.

Copy to Customer

Copies the selected voice prompt file form the Project Voiceprompts to the Customer Files.

Delete Voicefile

Removed the selected voice prompt file from the Customer files list and removes it from the system.

Browse

Use this button to find a voice file stored on your local workstation that you would like to upload to the project. When selected, the file will be displayed to the left of the button.

Upload

Instructs the browser to upload the displayed file from the workstation to the project.

Download

Used to download the selected voice prompt file from the Project server to the local workstation.

Delete

Deletes the selected voice prompt file from the Project server.

Livemsg

Converts the selected voice prompt file to the "live.vox" file to be used during processing. The "live.vox" file is played or each "live" answered call.

Machinemsg

Converts the selected voice prompt file to the "machine.vox" file to be used during processing. The "machine.vox" file is played or each "voicemail" or "answering machine" answered call.
