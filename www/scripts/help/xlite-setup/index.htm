X-Lite Setup

Download and install X-Lite from here.

Obtain your username and password from your provider.

Launch X-Lite and configure a SIP account by using the "SIP Account Settings..." menu option.

Enter YOUR username and password, as well as the Domain (216.66.234.212) and Domain Proxy (216.66.234.212:8060) as follows:

Press OK and return to the main phone UI. Your phone should register with the domain.

If you phone looks like it registered properly, you can try a test. Dial 111 (triple one) for an echo test. The switch should answer, "This is an echo test ...".
