The fields are as follows:

  1. Date

  2. Time

  3. Called Number

  4. Call duration (Raw, not the billing duration)

  5. Disposition

  6. Circuit reference

  7. Internal codes

  8. Related Prospect number in an Agent CDR.



Prospect Dispositions

HA, HU, HN: Live person answered

MA, MN: Answering machine answered

NA: No answer

BU: Busy

BA: Bad

FA: Fax


Agent Dispositions

AC: Agent connected

AB: Agent busy

AN: Agent no-answer

DA: Agent got dead air
