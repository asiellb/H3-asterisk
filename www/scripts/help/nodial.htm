Form Control Buttons

Browse

Use this button to find the Nodial List file using your local filesystem. When selected, the file will be displayed to the left of the button.

Upload

Instructs the browser to upload the displayed file from the workstation to the dialer.

Download

Used to download the selected file to the local workstation.

Delete

Removed the selected file from the Dial file list and removes it from the system.

Zip

Instructs the server to zip up the selected/highlighted file.
