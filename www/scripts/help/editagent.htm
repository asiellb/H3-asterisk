Edit Agent


The Agent Definition form is used to change or edit Agent information for "Press 1" type projects. Dialers will use the updated information in this form to process "Press 1" projects where an Agent is assigned.

Field Definitions

Name

A unique name to be used to reference all information regarding this specific Agent. Up to 20 alphanumaric characters including a-z, 0-9, dash("-") and underscore("_") characters can be used.

Password

Password used by Agent to sign onto the system. Once signed in, Agent can be activated to accept incoming calls from the dialers. Up to 20 alphanumaric characters including a-z, 0-9, dash("-") and underscore("_") characters can be used.

Email

Email address of Agent.

CallBack

Call back phone number for use by the Dialers to connect Agent with interested "Press 1" leads.

Project

Defines the active project Agent is assigned to receive processed call backs.

Status

Shows the current state of the Agent.

  • Active - Agent is active and ready.
  • Blocked - Agent is currently suspended or blocked from processing.
  • Signed Out - Agent is currently offline and un-available.

Lst Change

Displays the date and time of the last change to the Agent record.
