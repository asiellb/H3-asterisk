Customer Page

Menu Button Explanation

All Projects
Opens the project list page, showing all projects, blocked, old and current.
Active Projects
Opens the project list page, showing only current active projects.
DNC
Opens the Do Not Call (DNC) list maintenance page.
Payment
Users who are eligible to make credit card payments will see this button. It opens a page that takes credit card payment details to increase the customer credit balance.
Users
Opens the user maintenance page.
Agents
Opens the agent mainenance page.
Logoff
Logs the user off.

Info

Balance
Displays the available credit.
Authorized Agents
Number of agents that may log on simultaneously.
Time Zone
Displays the customer's time zone.

Generate a report

This generates a CSV file (that can be opened as a spreadsheet) showing project statistics for the supplied time period.
