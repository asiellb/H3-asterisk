Users

Users are people who can login like a customer (supervisor) can with optionally limited rights.

Users can be created with the follow levels of responsibility:

  1. Only View Own Projects - Can only view and manage the projects created under own user number.
  2. View Other Projects - Can view projects created by others, but cannot make changes to them. Can change only his/her own projects.
  3. Edit Other Projects - Can view and change projects created by others. (but cannot create other users)
  4. Supervisor - Can view manage and change any project for Customer account.

Users are managed at supervisor level only.
