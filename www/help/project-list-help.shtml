Project List Page

Menu Button Explanation

Back
Returns to the customer page
New
Create a new project
Edit
Make changes to the selected project in the list.
Remove
Remove the selected project in the list.
Agent Activity
Displays real time activity of the agents on the selected cold calling project.
Report
Displays daily history of dialing activity and costs of the selected project in the list.
Leads
Opens the leads page for uploading and maintaining lead files.
Voice
Opens the Voice file Management page for the selected project.
CDR
Opens the Call Detail Records page for the selected project.
Recordings
Displays the list of recordings made for the selected cold calling project.
Test Call
Opens the test call control page to make a test call using the currently selected, and defined project parameters
Pause/Resume
Blocks all active projects / Resumes previously paused projects
Support
Opens the support page for the selected project.

Columns in the list

Sel
Use this column to select a project as a context for the menu button.
? Support
This column, titled with a question mark, shows any support issues that maybe outstanding for the project. Clicking on this cell opens the support page for that project.
Run Info
The process that allocates lines to projects runs every few minutes and updates this column. (So it can be a few minutes behind reality) It displays a short status message on the project, if the message is not "Running" then it shows the reason why the project is not running.
