Simple Browser Popup Test

This is a simple test page that opens a new window. It can sometimes diagnose a browser configuration problem, in the case where the browser may be blocking popups.

If you did not see a page with pop up with the sentence "This page popped up." on it, then perhaps you have popup blocking on.
